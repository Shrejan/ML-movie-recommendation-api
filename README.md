# 📖 Overview

This project is a **Machine Learning-based Movie Recommendation System** that generates highly relevant movie recommendations based on a user's movie preferences.

Instead of relying on simple genre matching, every movie is converted into a **Transformer-generated embedding vector** that captures semantic relationships between movies. User preferences are represented as a weighted embedding, and recommendations are generated using **Cosine Similarity**.

The API is built with **FastAPI** and deployed on **Render** for production use.

# 📂 Dataset

The recommendation engine is trained using approximately **50,000 movies**.

Movie representations are generated by combining information from:

- 🎭 Genres
- ⭐ Ratings
- 🏷️ Tags

These features are transformed into high-dimensional embeddings for semantic similarity search.

# 🧠 Recommendation Pipeline

```text
User Ratings
      │
      ▼
Weighted User Embedding
      │
      ▼
Cosine Similarity Search
      │
      ▼
Top Similar Movies
      │
      ▼
Top 10 Recommendations
```

# 📡 API Endpoint

### POST `/rec`

Accepts a list of user-rated movies and returns personalized recommendations.

### Request Body

```json
[
  {
    "imdbId": 1375666,
    "rating": 5
  },
  {
    "imdbId": 8116692,
    "rating": 4
  }
]
```


